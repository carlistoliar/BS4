Breakpoints

<div class="reactiveBox"> </div> 
    
      /* Small devices (landscape phones) */
      @media (min-width: 576px) {
      ########## this will not work #########
        body{
          background: red;
        }
      ########## this will not work #########

      ############ this will work ###########
         .reactiveBox{
          background: red;
        }
      ############ this will work ###########
      }

      /* Medium devices (tablets) */
      @media (min-width: 768px) {
        body{
          background: blue;
        }
      }

      /* Large devices (desktops) */
      @media (min-width: 992px) {
        body{
          background: green;
        }
      }

      /* Extra large devices (large desktops) */
      @media (min-width: 1200px) {
        body{
          background: yellow;
        }
      }

@media css calls will NOT work on elements outside the component that would have to be done with javascript
use a service (if you want to implement changes in multiple components, use @HostListener or create an Observable and subscribe...

Floats & Fixed Positions

Floats

<div class=float-right (left, none">float ... </div>
Float left

Float right

Float none

Class float moves a div to left or right or none (clear)


Responsive Float

<div class="float-xx-xxxt>float...</div>
Float right on small or wider

Float right on medium or wider

Float right on large or wider

Float right on extra large or wider

Float left on small or wider

Float left on medium or wider

Float left on large or wider

Float left on extra large or wider


Float none on small or wider

Float none on medium or wider

Float none on large or wider

Float none on extra large or wider

Class float-sm (md, lg, xl) -right (left, none) responsively will push a div depending on device or browser width



<div class="clearfix">
<button class="float-left">float left</button>
<button class="float-right">float right</button>
</div>

Class clearfix ends the previous floating fo it does not affect items that come after


Fixed Top

<h3 class="fixed-top">Fixed top<'h3>

Class fixed-top will make the h3 (are any other structure) stick to the top of the page good for logos


Sticky

<h3 class="sticky-top">sticky-top<'h3>

Class sticky-top will make the h3 (are any other structure) stick to the top of the page when the user scrolls past that point and drop again if the user scroll down


Fixed Bottom

<h3 class="fixed-bottom">fixed-bottom<'h3>

Class fixed-bottom will make the h3 (are any other structure) stick to the bottom of the page good for footers
